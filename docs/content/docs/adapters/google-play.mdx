## Installation

```bash title="Terminal"
bun add @apollo-deploy/adapter-google-play
```

## Configuration

```typescript title="lib/integrations.ts" lineNumbers
import { createGooglePlayAdapter } from '@apollo-deploy/adapter-google-play';

const googlePlay = createGooglePlayAdapter({
  clientId: process.env.GOOGLE_CLIENT_ID!,
  clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
  // Optional: webhook endpoint shared secret
  webhookSecret: process.env.GOOGLE_PLAY_WEBHOOK_SECRET,
});
```

| Option | Type | Description |
|--------|------|-------------|
| `clientId` | `string` | Google OAuth 2.0 Client ID |
| `clientSecret` | `string` | Google OAuth 2.0 Client Secret |
| `webhookSecret` | `string?` | Shared secret for Pub/Sub notification verification |

## Environment variables

```bash title=".env"
GOOGLE_CLIENT_ID=1234567890-abc.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-...
```

## OAuth

Google Play uses OAuth 2.0 with long-lived refresh tokens. Access tokens expire after **1 hour**.

```typescript title="lib/google-play-oauth.ts" lineNumbers
const url = adapter.oauth.getAuthorizationUrl({
  state,
  scopes: ['https://www.googleapis.com/auth/androidpublisher'],
  redirectUri,
  extras: { access_type: 'offline', prompt: 'consent' }, // Required for refresh token
});

const tokens = await adapter.oauth.exchangeCode({ code, redirectUri });
// tokens.providerData.email — the Google account email

const refreshed = await adapter.oauth.refreshToken(tokens.refreshToken!);
```

<Callout type="info">
Always pass `access_type: 'offline'` and `prompt: 'consent'` in `extras` to ensure Google returns a refresh token.
</Callout>

## Real-time notifications

Google Play delivers real-time developer notifications via Cloud Pub/Sub. Configure a push subscription pointing to your `/webhooks/google-play` endpoint.

| Notification type | Normalized `eventType` |
|---|---|
| Subscription purchased | `subscription.purchased` |
| Subscription renewed | `subscription.renewed` |
| Subscription cancelled | `subscription.cancelled` |
| Subscription expired | `subscription.expired` |
| One-time product purchased | `purchase.completed` |

## Token metadata

| Field | Value |
|-------|-------|
| `expiresInSeconds` | `3600` (1 hour) |
| `refreshable` | `true` |
| `rotatesRefreshToken` | `false` |
| `requiresRefreshLock` | `false` |
