## Installation

```bash title="Terminal"
bun add @apollo-deploy/adapter-slack
```

## Configuration

```typescript title="lib/integrations.ts" lineNumbers
import { createSlackAdapter } from '@apollo-deploy/adapter-slack';

const slack = createSlackAdapter({
  clientId: process.env.SLACK_CLIENT_ID!,
  clientSecret: process.env.SLACK_CLIENT_SECRET!,
  signingSecret: process.env.SLACK_SIGNING_SECRET!,
});
```

| Option | Type | Description |
|--------|------|-------------|
| `clientId` | `string` | Slack app Client ID |
| `clientSecret` | `string` | Slack app Client Secret |
| `signingSecret` | `string` | Signing secret for `X-Slack-Signature` verification |

## Environment variables

```bash title=".env"
SLACK_CLIENT_ID=123456.789
SLACK_CLIENT_SECRET=abc123
SLACK_SIGNING_SECRET=signing_secret
```

## OAuth

Slack uses OAuth 2.0. The `botToken` and `teamId` are returned in `tokens.providerData`:

```typescript title="lib/slack-oauth.ts" lineNumbers
const url = adapter.oauth.getAuthorizationUrl({
  state,
  scopes: ['channels:read', 'chat:write', 'channels:history'],
  redirectUri,
});

const tokens = await adapter.oauth.exchangeCode({ code, redirectUri });
// tokens.providerData.teamId
// tokens.providerData.teamName
// tokens.providerData.botUserId
```

<Callout type="info">
Slack bot tokens **never expire** unless token rotation is explicitly enabled. `expiresInSeconds` is `null`.
</Callout>

## Webhook events

Slack sends a signed `X-Slack-Signature` (HMAC-SHA256 over `v0:{timestamp}:{rawBody}`).

The adapter automatically handles the `url_verification` challenge synchronously:

```typescript title="app/webhooks/slack.ts" lineNumbers
// No special handling needed — the hub returns the challenge inline
const response = await hub.webhooks.slack(req);
// 200 { challenge: "..." } ← hub handles this before async dispatch
```

| Slack event | Normalized `eventType` |
|---|---|
| `message` | `message.created` |
| `message` (edited) | `message.updated` |
| `reaction_added` | `reaction.added` |
| `reaction_removed` | `reaction.removed` |
| `member_joined_channel` | `channel.member_joined` |

## Token metadata

| Field | Value |
|-------|-------|
| `expiresInSeconds` | `null` (never) |
| `refreshable` | `false` |
| `rotatesRefreshToken` | `false` |
| `requiresRefreshLock` | `false` |
