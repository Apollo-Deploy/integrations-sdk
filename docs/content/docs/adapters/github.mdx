## Installation

```bash title="Terminal"
bun add @apollo-deploy/adapter-github
```

## Configuration

```typescript title="lib/integrations.ts" lineNumbers
import { createGithubAdapter } from '@apollo-deploy/adapter-github';

const github = createGithubAdapter({
  appId: process.env.GITHUB_APP_ID!,
  privateKey: process.env.GITHUB_PRIVATE_KEY!,
  webhookSecret: process.env.GITHUB_WEBHOOK_SECRET!,
  // Optional: GitHub Enterprise Server
  baseUrl: process.env.GITHUB_BASE_URL,
});
```

| Option | Type | Description |
|--------|------|-------------|
| `appId` | `string` | GitHub App ID from your app's settings page |
| `privateKey` | `string` | PEM-encoded RSA private key (include newlines) |
| `webhookSecret` | `string` | The secret configured in your app's webhook settings |
| `baseUrl` | `string?` | GitHub Enterprise Server base URL (default: `https://api.github.com`) |

## OAuth / App installation

GitHub Apps use installation-based access tokens rather than standard OAuth refresh tokens. The `installationId` is stored in `tokens.providerData`:

```typescript title="lib/github-oauth.ts" lineNumbers
// Get authorization URL — opens GitHub's app installation page
const url = adapter.oauth.getAuthorizationUrl({ state, scopes: [], redirectUri });

// Exchange code after GitHub redirects back
const tokens = await adapter.oauth.exchangeCode({ code, redirectUri });
// tokens.providerData.installationId — use this for all API calls

// "Refresh" = generate new JWT + hit /app/installations/{id}/access_tokens
const refreshed = await adapter.oauth.refreshToken(tokens.accessToken);
```

<Callout type="info">
GitHub installation tokens expire after **1 hour** and are refreshed by re-signing a JWT with your private key — `rotatesRefreshToken` is `false`.
</Callout>

## Webhook events

GitHub sends `X-Hub-Signature-256` HMAC-SHA256 signatures. The adapter verifies these automatically.

Supported normalized event types:

| GitHub event | Normalized `eventType` |
|-------------|------------------------|
| `push` | `push` |
| `pull_request` (opened/closed/merged) | `pull_request.opened` / `pull_request.closed` / `pull_request.merged` |
| `pull_request_review` | `pull_request.reviewed` |
| `issues` | `issue.created` / `issue.updated` / `issue.closed` |
| `check_run` | `check_run.completed` |
| `deployment_status` | `deployment.status_changed` |
| `release` | `release.published` |

## Token metadata

| Field | Value |
|-------|-------|
| `expiresInSeconds` | `3600` (1 hour) |
| `refreshable` | `true` |
| `rotatesRefreshToken` | `false` |
| `requiresRefreshLock` | `false` |
