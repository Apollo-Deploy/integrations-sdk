## Lifecycle

### `initialize()`

```typescript
hub.initialize(): Promise<void>
```

Calls `onRegister` then `onReady` on every adapter in registration order. Throws if any lifecycle hook throws. Must be called before `hub.webhooks` or capability methods are used.

### `shutdown()`

```typescript
hub.shutdown(): Promise<void>
```

Calls `onShutdown` on all adapters concurrently (using `Promise.allSettled`). Individual shutdown errors are logged but do not prevent other adapters from shutting down.

## Webhook routing

### `webhooks`

```typescript
hub.webhooks: WebhookRouter<TAdapters>
```

A type-safe Proxy where `hub.webhooks[key]` only compiles if `key` is a registered adapter key.

```typescript
hub.webhooks.github(request, options)  // ✓ — 'github' is registered
hub.webhooks.notion(request, options)  // ✗ — compile error if 'notion' is not in TAdapters
```

Each call:
1. Verifies the webhook signature
2. Handles synchronous challenges (Slack `url_verification`)
3. Returns `200` immediately
4. Dispatches the normalized event via `waitUntil` (async)

```typescript
const response = await hub.webhooks.github(
  { rawBody, headers, body },
  { secret: process.env.GITHUB_WEBHOOK_SECRET!, waitUntil: ctx.waitUntil },
);
// response: { statusCode: 200, body?: unknown }
```

## Adapter access

### `getAdapter(key)`

```typescript
hub.getAdapter<K extends keyof TAdapters>(key: K): TAdapters[K]
```

Returns the adapter registered under `key`. Throws [`UnknownAdapterError`](/docs/api/errors#unknownadaptererror) if not found.

### `getAdaptersByCapability(capability)`

```typescript
hub.getAdaptersByCapability(capability: AdapterCapability): IntegrationAdapter[]
```

Returns all adapters that declared the given capability. Useful for broadcasting to every provider of a type.

### `getRegisteredAdapters()`

```typescript
hub.getRegisteredAdapters(): string[]
```

Returns an array of registered adapter keys.

### `listAdapters()`

```typescript
hub.listAdapters(): AdapterInfo[]
```

Returns a snapshot of static information for every registered adapter. Useful for building integration listing UIs.

**`AdapterInfo`**

## Event subscriptions

### `onEvent(type, handler)`

```typescript
hub.onEvent(eventTypeOrTypes: string | string[], handler: EventHandler): void
```

Register a handler for one or more event types. Fires when any adapter emits a matching event.

```typescript
hub.onEvent('push', async (event, ctx) => { ... });
hub.onEvent(['issue.created', 'issue.updated'], async (event, ctx) => { ... });
```

### `onAnyEvent(handler)`

```typescript
hub.onAnyEvent(handler: EventHandler): void
```

Wildcard handler — fires for every event from every adapter.
