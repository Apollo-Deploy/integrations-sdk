## High-level API

### `createCryptoProviderFromEnv()`

```typescript
createCryptoProviderFromEnv(): CryptoProvider
```

Creates a `CryptoProvider` by reading `KMS_ROOT_KEY_B64` from `process.env`. Throws `CryptoConfigError` if the variable is missing or the key is not valid base64.

```bash title=".env"
KMS_ROOT_KEY_B64=<base64-encoded 32-byte key>
```

### `createCryptoProvider(rootKey)`

```typescript
createCryptoProvider(rootKey: Buffer): CryptoProvider
```

Creates a `CryptoProvider` from an explicit 32-byte root key buffer.

### `getRootKeyFromEnv()`

```typescript
getRootKeyFromEnv(): Buffer
```

Reads and decodes `KMS_ROOT_KEY_B64`. Useful when you need the raw key for other operations.

## `CryptoProvider`

The `CryptoProvider` interface provides high-level encrypt/decrypt with context-bound key derivation.

```typescript
interface CryptoProvider {
  encrypt(plaintext: string, context: EncryptionContext): Promise<EncryptedEnvelope>;
  decrypt(envelope: EncryptedEnvelope, context: EncryptionContext): Promise<string>;
}
```

### `encrypt(plaintext, context)`

Derives a data key from the root key using HKDF-SHA256 bound to `context`, then encrypts with AES-256-GCM.

```typescript title="lib/tokens.ts" lineNumbers
const crypto = createCryptoProviderFromEnv();

const envelope = await crypto.encrypt(accessToken, {
  orgId: 'org_123',
  entityId: 'user_456',
  purpose: 'oauth-token',
});

// Persist envelope in your database
await db.connections.update({ encryptedAccessToken: envelope });
```

### `decrypt(envelope, context)`

Re-derives the data key from the same context and decrypts. Throws `DecryptError` if the auth tag fails (wrong key or tampered ciphertext).

```typescript title="lib/tokens.ts" lineNumbers
const accessToken = await crypto.decrypt(
  connection.encryptedAccessToken,
  { orgId: 'org_123', entityId: 'user_456', purpose: 'oauth-token' },
);
```

## Low-level API

Use the low-level functions only when you need fine-grained control.

### `encryptGCM(key, plaintext)`

```typescript
encryptGCM(key: Buffer, plaintext: string): Promise<EncryptedEnvelope>
```

Encrypts `plaintext` with the provided 32-byte AES-256-GCM key.

### `decryptGCM(key, envelope)`

```typescript
decryptGCM(key: Buffer, envelope: EncryptedEnvelope): Promise<string>
```

Decrypts an envelope with the provided key. Throws `DecryptError` on auth tag mismatch.

### `deriveDataKey(rootKey, context)`

```typescript
deriveDataKey(rootKey: Buffer, context: EncryptionContext): Promise<Buffer>
```

Derives a 32-byte data key using HKDF-SHA256. The `EncryptionContext` fields are used as HKDF info, binding the derived key to a specific tenant/entity/purpose.

### `zeroize(buffer)`

```typescript
zeroize(buffer: Buffer): void
```

Overwrites a key buffer with zeros to prevent it from lingering in memory. Call this after you're done with a derived key:

```typescript lineNumbers
const dataKey = await deriveDataKey(rootKey, context);
try {
  const envelope = await encryptGCM(dataKey, plaintext);
  return envelope;
} finally {
  zeroize(dataKey);
}
```
